1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. Const brukes når verdien på variablene ikke skal endres senere i programmet. I disse oppgavene er det mulig å bruke const på alt.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parenteser endre regnerekkefølgen. Programmet regner ut det som er inni parentesen først. Dette kan endre resultat på et regnestykket.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. Vi kan bruke too.Fixed(antall desimaler) for å velge hvor mange desimaler vi vil ha med. Vi kan også bruke Math.floor(tallet) for å fjerne alle desimaler.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. For å samenligne to flyttall kan vi bruke epsilon: Math.abs(12.199999 - 12.2) < 0.0001

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Sammenlignsopperatører: (==, >, <, =!) Const paastand = (5 == 3) Ved å skrive slike påstander vil programmet skrive ut om de er falske eller sanne.

  6. How does the Math.random() function work, and what can it be used for?

    7. Math.random() generer et tilfeldig tall innenfor den rammen du gir den. Det kan brukes til alt som krever tilfeldig generering av tall som blant annet spill, simuleringer eller tilfeldige utvalg.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. (%) gir restene av en heltalsdivisjon. Fks: (10 % 3 = 1) fordi 3*3=9 og 10-9=1. Dette kan brukes til å finne ut om et tall er ett partall, finne rester etter du har foretatt et regnestykke.
