    4. - Innkapsling betyr at vi skjuler detaljer inni en klasse, og kun gjør det vi ønsker tilgjengelig for andre. Det gjør at brukere av klassen trenger bare å vite hvordan de kaller metodene, ikke hvordan de fungerer inni. Hvordan lage private og public medlemmer i en konstruktørfunksjon: function BankAccount(initialBalance) { let balance = initialBalance; // private this.deposit = function(amount) { // public if(amount > 0) balance += amount; }; this.getBalance = function() { // public return balance; }; } • balance er privat – kan ikke endres direkte utenfor funksjonen. • deposit og getBalance er public – kan brukes av koden utenfor objektet.
